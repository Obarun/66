execl-subuidgid

This command substitute a litteral GID UID term by the UID GID of the current owner of the process.

Interface 

	execl-subuidgid [ -o owner ] prog
  • execl-subuidgid reads the command and substitute the variable UID GID on prog.

Options

  • -o owner : set UID GID of owner instead of the current one.

Usage examples 

	execl-subuidgid
	if { mkdir -p /run/user }
	chown -R $UID:$GID /run/user 
	
	execl-subuidgid -o root
	if { mkdir /run }
	chmow -R $UID:$GID /run/user
