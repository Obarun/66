execl-cmdline

This command parse a command line into an execline script.

Interface

In an execlineb script: 

	execl-cmdline -S { command }
  • execl-cmdline reads the command and remove all space or 'no word' found before executing it.
  • execl-cmdline will exit with the exit code of the command.

Options

  • -S  : split into separate words a command considered as one word by execline scripting language.

Note and usage examples

execl-cmdline should be used at the end of the execline scripts. If you want run another program after the call of execl-cmdline you need to use e.g if command, foreground, background, ...

The following command: 

	execl-cmdline { /usr/bin/ntpd -d "" -S }
will give: 
	/usr/bin/ntpd "-d" "-S"

The following command: 

	execl-cmdline -S { /usr/bin/ntpd "-d -S" }
will give: 
	/usr/bin/ntpd "-d" "-S"

To run another program after the call of execl-cmdline program: 

	foreground { execl-cmdline { /usr/bin/ntpd -d "" -S } }
